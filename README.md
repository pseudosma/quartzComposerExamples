#  Quartz Composer Examples

A few examples of compositions from the now deprecated Quartz Compozer.

## Audio Bar

A classic audio bar for visualizing sound input.

## Audio Bounce

A visualization of sound input using a bouncing dot that moves across the screen. 

## Fade Audio Bars

An expansion on the audio bar that includes a fade effect.

## Movement With Limits By Keys (Broken)

Demonstrates movement to only the bounds of the presentation using the arrow keys as input. Currently in a broken state.

## Snowflakes

A simple display of snowflakes falling.

## Sound Line

Another clasic audio visualization using fluctuations in a line to display sound.

## Sound Flash

Demonstrates audio input triggering changes in screen brightness.
